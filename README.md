# Peaceful Minds Therapy — Marketing Landing Page

A professional, conversion-focused landing page designed to attract therapy clients.

## What's included

`index.html` — a fully self-contained, responsive landing page with:

- **Hero section** — compelling headline, sub-copy, and a prominent free-consultation CTA
- **Services grid** — Anxiety & Stress, Depression, Couples Therapy, Trauma/PTSD, Life Transitions, Child & Teen Therapy
- **About section** — therapist bio with credentials and trust signals
- **How it works** — four-step process to lower the barrier to entry
- **Testimonials** — three authentic client stories
- **FAQ** — six commonly asked questions (cost, online sessions, confidentiality, etc.)
- **CTA section** — email and phone call-to-action
- **Footer** — contact details and accreditation info

## How to use

Open `index.html` in any modern browser — no build step or dependencies required.

```bash
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Or serve it with any static host (GitHub Pages, Netlify, Vercel, etc.).

## Design decisions that attract clients

| Feature | Rationale |
|---|---|
| Free 20-min consultation CTA | Removes financial risk — the #1 barrier to starting therapy |
| Trust strip (BACP, confidentiality, stars) | Builds credibility above the fold |
| Warm sage-green palette | Communicates calm, safety, and nature |
| Plain-English FAQ | Addresses objections before they become drop-offs |
| Sliding-scale fee mention | Signals accessibility and inclusivity |
| Online + in-person options | Widens the addressable audience |
